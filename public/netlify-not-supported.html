EspoCRM needs a PHP runtime

This deploy preview is hosted on Netlify, which is optimized for static sites and serverless JavaScript functions. EspoCRM is a PHP application that also requires a supported database such as MySQL, MariaDB, or PostgreSQL. Because Netlify does not provide that runtime, the application cannot run here.

You can review the Netlify deployment notes for guidance on hosting EspoCRM with compatible infrastructure.

Supported options include:

  • Using a VPS or hosting provider that offers PHP 8.2–8.4 together with a database.
  • Deploying Docker images or other scripted installations described in the EspoCRM documentation.
  • Running EspoCRM locally for evaluation before promoting it to production hosting.

The static assets in this preview only show this notice so visitors are not presented with a confusing 404 page.
