EspoCRM needs a PHP runtime

You are viewing a Netlify deploy of the EspoCRM repository. Netlify hosts static assets and serverless JavaScript functions, but the CRM application relies on PHP 8.2–8.4 and a database such as MySQL, MariaDB, or PostgreSQL. Because Netlify cannot execute PHP code, the full application cannot run here.

This compatibility page is published to clarify the limitation and prevent confusing 404 responses. The complete deployment instructions for supported environments are described in our documentation.
